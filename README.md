# Java Practice & Learning Repository
[![Ask DeepWiki](https://devin.ai/assets/askdeepwiki.png)](https://deepwiki.com/selvin-r/IDE-IN-JAVA)

This repository serves as a personal collection of Java code, documenting a learning journey through various programming concepts, data structures, algorithms, and problem-solving exercises. It includes solutions to challenges from platforms like LeetCode and HackerRank, implementations of fundamental computer science concepts, and small-scale application examples.

## Getting Started

This repository is a collection of individual Java files. There is no central build file (`pom.xml` or `build.gradle`). To run a specific piece of code:

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/selvin-r/ide-in-java.git
    ```
2.  **Navigate to the source directory:**
    ```sh
    cd ide-in-java/src
    ```
3.  **Compile the desired Java file.** For example, to compile the Tic-Tac-Toe game:
    ```sh
    javac TicTacToe.java
    ```
4.  **Run the compiled class:**
    ```sh
    java TicTacToe
    ```
    You can follow the same process for any other file containing a `main` method.
